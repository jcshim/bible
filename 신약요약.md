# 1️⃣ 신약성경의 전체 구조 (핵심 3단계)

신약성경은 **구원의 세 단계**를 보여준다.

1. **복음서 → 칭의 (Justification)**

   * 예수 그리스도를 통해 하나님 나라가 시작됨
   * 죄인이 의롭게 되는 구원의 시작

2. **서신서 → 성화 (Sanctification)**

   * 구원받은 성도가 어떻게 살아야 하는지 가르침
   * 교회와 신앙생활의 원리 제시

3. **요한계시록 → 영화 (Glorification)**

   * 하나님 나라의 완성
   * 새 하늘과 새 땅, 최종 승리

👉 즉
**구원 = 칭의 → 성화 → 영화**
